⚙ Instellingen — Google Sheets koppeling
Hoe werkt dit?
  1. Maak een Google Sheet aan en open Extensies → Apps Script
  2. Plak het Apps Script (zie hieronder) en klik Implementeren → Nieuwe implementatie
  3. Kies type: Web-app, uitvoeren als: Ik, toegang: Iedereen
  4. Kopieer de gegenereerde Web App URL en plak die hieronder
  5. Upload dit HTML-bestand naar GitHub Pages en deel de URL met je collega's
